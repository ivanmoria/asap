SISTEMA DE CONTROLE

Configure a base harmônica para os osciladores.

O Harmonic Lab processa 28 variações de acordes em tempo real. Utilize os controles do cabeçalho para manipular a saída:

  • • POWER: Ativa ou corta o fluxo de áudio global dos osciladores.
  • • TÔNICA & OITAVA: Selecione a nota base no teclado e ajuste a altura nos botões [+] e [-].
  • • ENARMONIA: Toque no Display de Frequência (Hz) no cabeçalho para alternar entre sustenidos (♯) e bemóis (♭).
  • • TIPO DE ACORDE: Escolha a estrutura harmônica no grid superior de botões.
Dica:

A enarmonia é inteligente e sugere o melhor padrão para a tônica escolhida, mas você tem o controle manual total ao tocar no display central.
